# Anthropic Academy — Root Context

This is a learning repo containing independent coursework from Anthropic Academy.

## Structure
Each subdirectory is a **separate, independent course project**. Do not assume
conventions, dependencies, or patterns from one course folder apply to another.
Always treat each course folder as its own isolated environment.

## General Rules
- Never commit `.env` files or real API keys
- Each course may use a different language/stack — check that course's README first
- When working inside a course folder, that folder's CLAUDE.md takes priority
