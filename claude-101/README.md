## 📍What This Course Covers

- Claude's core principles and capabilities
- Prompting: setting context, defining the task, and specifying format

- Projects, Artifacts, Skills, Connectors: usage and applications

- Claude platforms and applications

## 📚 Takeaways

- **Delegation-Diligence loop**. Delegate a task to Claude, then evaluate the output. Decide how to sharpen the prompt and enhance Claude output while discovering what tasks need more human involvement.
- **Projects and context.** A well-configured project with a knowledge base and clear instructions paired with the use of skills provides a clean structure for long-term workflows.
- **Claude Modes.** Knowing when to use research mode (comprehensive synthesis) vs. web search (quick facts) vs. extended thinking (reasoning without external data) is a meaningful distinction for getting the right help.
- **Connectors.** The MCP-based connector system lets Claude interact directly with external tools, local files, and services, boosting workflow productivity significantly.
